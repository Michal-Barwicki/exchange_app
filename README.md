# Exchange APP,
In this project user can create account and then login to main panel where he can exchange his
 currencies.
 
## Installation

1. Clone the repo,
2. Instal npm packages

### `npm install`

3. Create a production Build

### `npm run build`

4. Run index.js

### `node run index.js`

5.  When the server is started, visit

### `http://localhost:5000/`
on your browser
